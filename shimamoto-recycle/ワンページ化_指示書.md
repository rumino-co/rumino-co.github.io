# リサイクルセンターHP ワンページ化 指示書

親会社サイト（嶋本建設）を2026-06-12にワンページ化したのと同じ要領で、リサイクルセンターHPも「トップから縦スクロールで全部見れる」形にするための指示書。

**親会社の完成版を参考にする:** `G:\マイドライブ\06　会社\嶋本建設\ホームページ\index.html` と `css\style.css`

---

## ゴール

トップページ（index.html）を下にスクロールするだけで、取扱品目・会社案内・お知らせまで全部見れる形にする。ヘッダーのタブ（トップ/取扱品目/会社案内/お知らせ/お問い合わせ）を押すと、同じページ内のそのセクションへスーッとスクロールする。各セクションに「詳しく見る →」ボタンを付け、深く見たい人は専用ページ（service.html等）へ飛べる。

**専用ページ（service.html / profile.html / news.html / contact.html）は残す**（消さない）。SEOとお知らせ増加への対応のため。

## 手順

### Step 1: ナビゲーションをアンカー化

index.html のヘッダー内ナビと、ロゴ、フッターナビのリンク先を専用ページ → ページ内アンカーに変更:

| 変更前 | 変更後 |
|---|---|
| `href="index.html"`（ロゴ・トップ） | `href="#top"` |
| `href="service.html"`（取扱品目） | `href="#items"` |
| `href="profile.html"`（会社案内） | `href="#profile"` |
| `href="news.html"`（お知らせ） | `href="#news"` |
| `href="contact.html"`（お問い合わせ） | `href="#contact"` |

⚠️ ナビのアンカー化は **index.html だけ**。service.html等の専用ページ側のナビは `index.html` 等の通常リンクのまま（別ページから戻れるように）。

### Step 2: 各セクションにIDを付与（index.html）

- ヒーロー `<section class="hero">` → `<section class="hero" id="top">`
- 取扱品目 `<section class="section section-gray">`（items-grid のあるやつ）→ `id="items"` を追加
- お問い合わせ `<section class="contact-section">` → `id="contact"` を追加
- 新規追加する会社案内セクション → `id="profile"`
- 新規追加するお知らせセクション → `id="news"`

### Step 3: 会社案内の要約セクションを追加（index.html）

profile.html から会社概要テーブル（`.company-table`）の主要項目をコピーして、取扱品目セクションの後あたりに挿入。`id="profile"` を付ける。末尾に「会社案内・アクセスを見る →」の `.more-btn` ボタン（リンク先 `profile.html`）。

掲載項目の例（profile.html の実際の値に合わせる）:
- 会社名: 嶋本建設株式会社リサイクルセンター
- 所在地: 熊本県上益城郡益城町杉堂690番地3
- TEL / FAX: 096-289-8550 / 096-289-8551
- 営業時間: 8:00〜17:00
- 許可: 産業廃棄物処分業許可・一般廃棄物処分業許可 など

### Step 4: お知らせの要約セクションを追加（index.html）

news.html から最新のお知らせ（`.news-list`）を2〜3件コピーして挿入。`id="news"` を付ける。末尾に「お知らせ一覧を見る →」の `.more-btn` ボタン（リンク先 `news.html`）。

### Step 5: 既存の「取扱品目の詳細を見る」ボタンを `.more-btn` に統一（任意）

現状は `.contact-btn` を流用しているが、親会社と見た目を揃えるなら `.section-more` ＋ `.more-btn` 構造に置き換える。

```html
<div class="section-more">
  <a href="service.html" class="more-btn">取扱品目を詳しく見る
    <svg viewBox="0 0 24 24" width="16" height="16" fill="none" stroke="currentColor" stroke-width="2.5"><path d="M5 12h14M13 5l7 7-7 7"/></svg>
  </a>
</div>
```

### Step 6: CSSに以下を追加（css/style.css）

親会社の style.css と同じものを追加する。`/* --- Greeting Page --- */` や任意の場所の前に挿入:

```css
/* --- Anchor scroll offset (fixed header) --- */
section[id], .contact-section[id] {
  scroll-margin-top: 80px;
}

/* --- Section "詳しく見る" button --- */
.section-more {
  text-align: center;
  margin-top: 40px;
}

.more-btn {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  padding: 13px 32px;
  border: 2px solid var(--primary);
  color: var(--primary);
  border-radius: 999px;
  font-size: 15px;
  font-weight: 600;
  transition: all 0.3s;
}

.more-btn:hover {
  background: var(--primary);
  color: var(--white);
}

.more-btn svg {
  transition: transform 0.3s;
}

.more-btn:hover svg {
  transform: translateX(4px);
}
```

そして、モバイル用に `@media (max-width: 768px) {` ブロックの中に追加:

```css
  section[id], .contact-section[id] {
    scroll-margin-top: 64px;
  }
```

⚠️ `html { scroll-behavior: smooth; }` は既にあるはず（スムーズスクロール用）。なければ追加。

### Step 7: CSSキャッシュバージョンを上げる

全HTMLの `<link rel="stylesheet" href="css/style.css?v=N">` の N を +1（例: v=1 → v=2）。index.html だけでなく全ページ揃える。

### Step 8: ローカル確認

```bash
cd "G:/マイドライブ/06　会社/嶋本建設リサイクルセンター/HP"
python -m http.server 8897
```

ブラウザで `http://localhost:8897/` を開き、タブを押してスクロールするか、固定ヘッダーに隠れないか確認。

### Step 9: GitHubにアップロード

親リポジトリ `rumino-co/rumino-co.github.io` の **`shimamoto-recycle/` フォルダ** に、HP内の全ファイルをアップロード（Add file → Upload files）。Cloudflareが自動デプロイ。

⚠️ アップロード先を間違えない。ルートではなく `shimamoto-recycle/` サブフォルダ。

### Step 10: 本番確認

`https://shimamoto-kensetsu.co.jp/shimamoto-recycle/` を開いて動作確認。会社のネットワーク（FortiGuard）からは403になることがあるので、その場合はスマホで確認。

## 親会社との違い・注意点

- リサイクルセンターは **TEL が違う**（096-289-8550、親会社は096-369-2048）
- 所在地が違う（益城町杉堂690番地3）

- canonical / og:url は `https://shimamoto-kensetsu.co.jp/shimamoto-recycle/...` のまま（変更不要）
- フッターの著作権は「リサイクルセンター」名義のまま
